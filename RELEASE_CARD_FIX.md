# Исправление дизайна карточки релиза

## Изменения по референсу

### Основные улучшения:

1. **Более компактная структура**
   - Уменьшены размеры бейджей (оценка и статус)
   - Оптимизированы отступы между элементами
   - Информация теперь в одну строку с разделителями "•"

2. **Жанры**
   - Отображаются отдельными тегами с #
   - Максимум 2 жанра для компактности
   - Жанры теперь flex-элементы, а не одна строка

3. **Информационная строка**
   - Все данные в одной строке: `Категория • Статус • Эпизоды • Год`
   - Использованы разделители "•" вместо отдельных бейджей
   - Единый стиль текста для всей информации

4. **Бейджи**
   - Уменьшены размеры (padding, font-size)
   - Более деликатные тени
   - Меньшие border-radius для современного вида

5. **Тени и визуальные эффекты**
   - Добавлена легкая тень на изображение
   - Улучшен градиент оверлея
   - Более плавные переходы

### Структура информации:

```
┌─────────────────┐
│   [Изображение] │ ← Бейдж оценки (★ 4.9) справа вверху
│                 │   Бейдж статуса внизу справа
└─────────────────┘
Название релиза
(2 строки максимум)

#жанр1 #жанр2

Категория • Статус • 8 из 12 эп • Лето 2025 г.
```

### Адаптивность:

- **Desktop**: полные размеры
- **Tablet (≤768px)**: уменьшенные шрифты и отступы
- **Mobile (≤360px)**: минимальные размеры для старых устройств

### Изменённые файлы:

1. `src/ui/components/ReleaseCard/ReleaseCard.tsx`
   - Переделана структура JSX
   - Жанры теперь отображаются как отдельные элементы
   - Вся информация в одной строке с разделителями

2. `src/ui/components/ReleaseCard/ReleaseCard.module.css`
   - Уменьшены размеры бейджей
   - Новые стили для info_text и separator
   - Оптимизированы адаптивные брейкпоинты
   - Добавлена тень на изображение

### Ключевые улучшения:

✅ Компактный дизайн
✅ Легкая читаемость
✅ Современный внешний вид
✅ Точное следование референсу
✅ Полная адаптивность
✅ Оптимизированная производительность
