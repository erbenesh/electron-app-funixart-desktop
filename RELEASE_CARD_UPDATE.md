# Обновление дизайна карточки релиза

## Изменения

### Компонент ReleaseCard

Карточка релиза была полностью переработана в соответствии с современными референсами дизайна:

#### Структура компонента

**Было:**
- Изображение с градиентным оверлеем снизу
- Вся информация располагалась поверх изображения
- Тяжелая визуальная композиция с множеством текста

**Стало:**
- Изображение с минимальным оверлеем
- Бейдж с оценкой в правом верхнем углу изображения
- Бейдж со статусом в правом нижнем углу изображения (если есть)
- Весь основной контент (название, жанры, информация) вынесен под изображение
- Чистый, современный и читаемый дизайн

#### Визуальные элементы

1. **Бейдж оценки** (Grade Badge)
   - Расположение: правый верхний угол изображения
   - Цвет: золотисто-жёлтый (rgba(255, 193, 7, 0.95))
   - Содержит: звездочку ★ и числовую оценку
   - Показывается только если есть оценка

2. **Бейдж статуса** (Status Badge)
   - Расположение: правый нижний угол изображения
   - Цвет: динамический (зависит от статуса в списке пользователя)
   - Показывается только если релиз добавлен в список

3. **Контент карточки**
   - **Название**: крупное, жирное, 2 строки максимум
   - **Жанры**: с префиксом #, серым цветом, 3 жанра максимум
   - **Информационные бейджи**: категория и статус релиза
   - **Нижняя строка**: информация об эпизодах и год/сезон выпуска

#### Анимации и взаимодействие

- Плавное появление карточки с эффектом подъема
- Hover эффект (только на устройствах с поддержкой hover):
  - Карточка поднимается вверх
  - Изображение немного увеличивается (scale 1.05)
  - Название меняет цвет на золотистый
  - Добавляется тень
- Active эффект: легкое уменьшение при нажатии

#### Адаптивность

Карточка полностью адаптивна для различных размеров экрана:
- Desktop: полный размер элементов
- Tablet (≤768px): уменьшенные шрифты и отступы
- Mobile (≤360px): ещё более компактный вид

### MediaCard компонент

Добавлена поддержка `topOverlay`:

```typescript
interface MediaCardProps {
  topOverlay?: React.ReactNode; // Новое свойство
  // ... остальные свойства
}
```

Это позволяет размещать контент в верхней части изображения (например, бейджи с оценками).

Все изменения обратно совместимы. Компонент ReleaseCard принимает те же пропсы:

```typescript
interface ReleaseCardProps {
    release: Release;
    clickCallBack?: (value: string) => void;
}
```

## Преимущества нового дизайна

1. **Улучшенная читаемость** - текст теперь не накладывается на изображение
2. **Современный вид** - дизайн соответствует актуальным трендам UI/UX
3. **Лучшая иерархия информации** - важная информация выделена
4. **Производительность** - убраны тяжелые эффекты и градиенты
5. **Мобильная оптимизация** - лучше работает на маленьких экранах

## Скриншоты

Новый дизайн основан на референсах:
- Карточки аниме с чистой структурой
- Информация разделена на логические блоки
- Акценты на ключевых элементах (оценка, статус)
